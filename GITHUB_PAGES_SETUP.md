# Инструкция: Как включить GitHub Pages

## Шаг 1: Откройте Settings
1. Перейдите на https://github.com/azimuthihi/Klever
2. Нажмите на вкладку **Settings** (в верхнем меню)

## Шаг 2: Найдите Pages
1. В левом меню прокрутите вниз до раздела **Code and automation**
2. Нажмите на **Pages**

## Шаг 3: Настройте источник
1. В разделе "Build and deployment"
2. В поле **Source** выберите: **Deploy from a branch**
3. В поле **Branch** выберите: **main**
4. В поле рядом (папка) выберите: **/ (root)**
5. Нажмите **Save**

## Шаг 4: Подождите
1. Обновите страницу через 1-2 минуты
2. Вверху появится зеленая плашка с текстом:
   "Your site is live at https://azimuthihi.github.io/Klever/"

## Шаг 5: Откройте сайт
Перейдите по ссылке: https://azimuthihi.github.io/Klever/

---

**Если возникнут проблемы**, пришлите скриншот страницы Settings → Pages
